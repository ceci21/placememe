Placememe

Placememe is a small tool to help you grab filler images to use for your website. Simply append to the URL the desired width and height of the filler image you want, like so: 

  http://placememe.herokuapp.com/500/600
  http://placememe.herokuapp.com/1600/1000

This app is my first app as part of my app-a-week project. You can find me at ceci.io and check out my blog for more info about my project.
